Raw Property Editor Plugin Readme

Overview

The Raw Property Editor plugin allows admins to edit existing or add new properties to users and groups.

Installation

Copy the rawpropertyeditor.jar into the plugins directory of your openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new rawpropertyeditor.jar file over the existing file.

Using the Plugin

The plugin is found under the users/groups tab of openfires admin panel and can be used after selecting a single user or group to edit properies of
